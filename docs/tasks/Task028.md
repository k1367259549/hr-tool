# API Endpoint

## Generate Monthly Review

```http
POST /api/review/monthly/generate
```

Request:

```json
{
  "year": 2026,
  "month": 1
}
```

# Monthly Review Output Schema

```json
{
  "summary": "string",
  "keyMetrics": "string",
  "majorAchievements": "string",
  "mainProblems": "string",
  "suggestions": "string",
  "nextMonthFocus": "string",
  "score": 85
}
```

# Rules

- Must use RecruitLog data within selected month
- Must calculate monthly KPI totals before AI call
- Must not invent missing data
- AI output must be valid JSON
- AI request must be logged through AIRequestLog
- Score must be between 0 and 100
