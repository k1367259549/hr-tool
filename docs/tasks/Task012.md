# API Endpoints

## Generate Review

```http
POST /api/review/generate
```

Request:

```json
{
  "date": "2026-01-01"
}
```

Behavior:

- Find RecruitLog by date
- Build prompt using `/prompts/review.md`
- Call AI service
- Parse JSON output
- Validate schema
- Save DailyReview
- Return saved review

## Get Review by Date

```http
GET /api/review/date/:date
```

---

# Review Output Schema

```json
{
  "summary": "string",
  "strengths": "string",
  "weaknesses": "string",
  "suggestions": "string",
  "score": 85
}
```

---

# Validation Rules

- score must be between 0 and 100
- all text fields are required
- log must exist before review generation
- duplicate review should update or replace existing review for the same log
